## Frontend (Admin Dashboard)

Next.js + Tailwind admin UI for:

- Dashboard overview
- Notification logs
- Template management
- Send test notification (plus AI tools)
- User preferences
- System metrics

### Run (Docker Compose)

From repo root:

```bash
docker compose up --build
```

UI: `http://localhost:3000`

### Config

See `.env.example` (`NEXT_PUBLIC_API_BASE_URL`).
